1. почему в java void main, в С int main

В Си при завершении main происходит уничтожение любых объектов с автоматическим сроком хранения,
параметр передается в std::exit

В Java когда JVM запускается, он запускает метод main()
но когда main() завершается, это не означает, что JVM завершает работу
JVM продолжает выполнять все потоки до тех пор, пока
1) не будет вызван Runtime.exit() ИЛИ 2) все нормальные (не демонические) потоки не завершатся.

3. что такое BigInteger, как он устроен внутри, зачем нужен

Для работы с целыми числами произвольного размера
которые превышают диапазон типов данных int или long.

Поддерживается диапазон от -2^2147483647 до 2^2147483647
Хранит знак и двоичное представление числа в массиве int[]
